# План реализации: модуль Auth

Дизайн: `docs/designs/2026-05-18-auth.md`

Auth — первая фича проекта, поэтому план включает скелет решения (Aspire AppHost,
ServiceDefaults, .NET API, Next.js-приложение). Скелет — Фаза 1; сама Auth-фича —
Фазы 2–4.

Скиллы для исполнителей: backend — `vertical-slice-architecture`, `id-generation`,
`error-handling`, `validation`, `component-testing`; frontend — `validation`,
`error-handling`. Конфигурация — только Options-паттерн; DTO наружу, не EF-сущности.
