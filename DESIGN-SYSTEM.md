# Дизайн-система

Простая система для согласованного UI. Любой новый экран или компонент строится **только** из описанных здесь токенов и паттернов — не вводи произвольные цвета, отступы или размеры.

Реализация: Tailwind CSS 4 + shadcn/ui. Токены задаются как CSS-переменные в `web/app/globals.css` и подключаются в `@theme`. В разметке используются семантические классы Tailwind (`bg-surface`, `text-muted` и т.п.), а не сырые значения.

## Принципы

1. **Семантика, а не значения.** Пиши `bg-danger`, а не `bg-red-600`. Цвет привязан к смыслу.
2. **Спокойный нейтральный фон, акцент — точечно.** Основной акцентный цвет используется для главных действий и активных состояний, не для декора.
3. **Один масштаб отступов и размеров.** Не изобретай промежуточные значения.
4. **Контент важнее хрома.** Минимум рамок и теней, разделяй пространством.

## Цвета

Палитра — нейтральная серая база + один акцент (indigo) + статусные цвета. Светлая и тёмная темы.

| Токен | Назначение | Light | Dark |
|-------|-----------|-------|------|
| `background` | Фон страницы | `#f8f9fb` | `#0e0f12` |
| `surface` | Карточки, панели, поповеры | `#ffffff` | `#191b1f` |
| `surface-muted` | Вторичные блоки, hover-фон | `#f1f2f5` | `#23262b` |
| `border` | Рамки, разделители | `#e4e6ea` | `#2e3138` |
| `foreground` | Основной текст | `#1a1c20` | `#f2f3f5` |
| `muted` | Вторичный текст, подписи | `#6b7280` | `#9aa1ad` |
| `primary` | Главные действия, ссылки, фокус | `#4f46e5` | `#6d66f0` |
| `primary-foreground` | Текст на `primary` | `#ffffff` | `#ffffff` |
| `danger` | Ошибки, деструктивные действия | `#dc2626` | `#f04444` |
| `success` | Успех, подтверждение | `#16a34a` | `#3ecf6b` |
| `warning` | Предупреждения | `#d97706` | `#f0a020` |

Состояния: hover — затемнение/осветление на ~8%, фокус — кольцо `ring-2 ring-primary` с `ring-offset-2`, disabled — `opacity-50` + `cursor-not-allowed`.

## Типографика

Шрифт: системный стек (`font-sans`). Моноширинный (`font-mono`) — для slug, ID, кода и diff.

| Класс | Размер / интерлиньяж | Применение |
|-------|---------------------|-----------|
| `text-2xl font-semibold` | 24 / 32 | Заголовок страницы |
| `text-lg font-semibold` | 18 / 28 | Заголовок секции, title карточки |
| `text-base` | 16 / 24 | Основной текст |
| `text-sm` | 14 / 20 | Вторичный текст, UI-контролы, таблицы |
| `text-xs font-medium` | 12 / 16 | Подписи, метки, бейджи |

Начертания: `font-normal` для текста, `font-medium` для UI-элементов, `font-semibold` для заголовков. Жирнее `semibold` не используем.

## Spacing и размеры

Шкала отступов (Tailwind по умолчанию, кратно 4px): `1 2 3 4 6 8 12 16`. Внутренние отступы компонентов — `2`–`4`, между блоками — `4`–`8`, между секциями — `8`–`12`.

- Радиус: `rounded-md` (6px) — контролы и карточки; `rounded-full` — бейджи и аватары. Других радиусов нет.
- Тени: `shadow-sm` — карточки и поповеры. Тяжелее не используем; глубину даёт `border`.
- Ширина контента: основной контейнер `max-w-5xl mx-auto px-4`.
- Высота интерактивных элементов: `h-9` (стандарт), `h-8` (компактный, в плотных списках).

## Компоненты

Базовые компоненты — из shadcn/ui, расположены в `web/components/ui`. Стилизуй их под токены выше; не подключай сторонние UI-киты.

### Button

| Вариант | Вид | Когда |
|---------|-----|-------|
| `primary` | `bg-primary text-primary-foreground` | Главное действие экрана (одно на экран) |
| `secondary` | `bg-surface border border-border` | Вторичные действия |
| `ghost` | прозрачный, `hover:bg-surface-muted` | Третичные действия, иконочные кнопки |
| `danger` | `bg-danger text-white` | Удаление, необратимые действия |

Высота `h-9`, `px-4`, `rounded-md`, `text-sm font-medium`. В состоянии загрузки — спиннер слева + `disabled`.

### Поля ввода

`Input`, `Textarea`, `Select`: `h-9` (textarea — авто), `bg-surface`, `border border-border`, `rounded-md`, `px-3`, фокус — кольцо `primary`. Ошибка — `border-danger` + текст ошибки `text-xs text-danger` под полем. Лейбл — `text-sm font-medium` сверху.

### Card

`bg-surface border border-border rounded-md p-4`. Заголовок `text-lg font-semibold`, тело — отступ `mt-2`.

### Badge

`rounded-full px-2 py-0.5 text-xs font-medium`. Используется для статусов, приоритетов и labels (см. ниже).

### Toast / уведомления

Короткие нотификации об операциях. Цвет фона — статусный (`success` / `danger`), позиция — снизу справа.

## Доменные токены

UI трекера должен показывать статус и приоритет Issue одинаково на всех экранах.

### Статус Issue

Бейдж с точкой-индикатором + подпись:

| Статус | Цвет точки | Стиль бейджа |
|--------|-----------|--------------|
| `backlog` | `muted` (серый) | `bg-surface-muted text-muted` |
| `todo` | синий `#3b82f6` | `bg-blue-50 text-blue-700` |
| `in-progress` | акцент `primary` | `bg-indigo-50 text-indigo-700` |
| `in-review` | `warning` | `bg-amber-50 text-amber-700` |
| `done` | `success` | `bg-green-50 text-green-700` |

### Приоритет Issue

Иконка-стрелка + подпись, цвет по уровню:

| Приоритет | Цвет |
|-----------|------|
| `low` | `muted` |
| `medium` | синий `#3b82f6` |
| `high` | `warning` |
| `urgent` | `danger` |

### Label

Бейдж, фон — из поля `Label.color` (хранится в БД), текст — авто-контраст (тёмный/светлый по яркости фона).

### AI-пометки

Контент, сгенерированный AI (`acceptance_criteria` с `ai_suggested: true`, AI-ревью), помечается значком-«искрой» и плашкой `bg-indigo-50 text-primary text-xs` «AI suggestion». Пока пользователь не принял — показывай кнопки «Accept / Edit / Rewrite».

## Состояния экранов

- **Loading** — скелетоны (`bg-surface-muted animate-pulse`) в форме контента, не спиннер по центру.
- **Empty** — иконка + короткий заголовок + одно действие (например, «No issues yet — Create issue»).
- **Error** — карточка с `danger`-акцентом, текст ошибки из `ProblemDetails` и кнопка повтора.

## Доступность

- Контраст текста ≥ 4.5:1 (для `muted` на `surface` проверено).
- Видимый фокус на всех интерактивных элементах — никогда не убирай `outline` без замены на `ring`.
- У иконочных кнопок — `aria-label`. У полей — связанный `<label>`.
- Статус/приоритет не передаются только цветом — всегда есть подпись или иконка.
